# Screenshots

Drop your Figma exports here before running the deck agent.

## Naming Convention

Use descriptive names so the agent knows what each screenshot shows:

```
feature-overview-01.png       ← Main feature UI
setup-step-1-navigate.png     ← Setup flow, step 1
setup-step-2-configure.png    ← Setup flow, step 2
feature-recordings.png        ← Specific capability: recordings
feature-analytics.png         ← Specific capability: analytics
```

## Tips

- Export at 2x resolution for sharp rendering in slides
- Crop to the relevant UI — no need for full browser chrome
- If showing a flow, number them: `setup-01.png`, `setup-02.png`, `setup-03.png`
- Light backgrounds work best on white content slides; dark UI screenshots go well on dark section slides
